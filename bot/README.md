# SpreadingPostsBot

Минимальный каркас Telegram-бота на aiogram v3 без бизнес-логики.

## Требования
- Python 3.11+

## Быстрый старт
1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Создайте файл `.env` по образцу `.env.example` и укажите токен бота.
3. Запустите бота:
   ```bash
   python -m bot.main
   ```
